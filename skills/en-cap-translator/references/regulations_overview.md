# Regulations Overview

Read this file only when the document needs domain context or terminology disambiguation.

## Euro NCAP

- Focus: consumer-facing vehicle safety assessment and ADAS test protocols.
- Common terms: AEB, LSS, SAS, VRU, occupant protection.
- Translation bias: favor practical test-language clarity over statutory phrasing.

## UN R79

- Focus: steering equipment requirements and steering automation functions.
- Common terms: CSF, ESF, ACSF, lateral acceleration limits, driver override.
- Translation bias: preserve requirement wording and function names exactly.

## UN R157

- Focus: Automated Lane Keeping System requirements.
- Common terms: ALKS, transition demand, minimum risk maneuver, operating envelope.
- Translation bias: keep operational constraints and fallback behavior precise.

## ISO 26262

- Focus: functional safety lifecycle for road vehicles.
- Common terms: hazard analysis, safety goal, ASIL, item definition, confirmation measures.
- Translation bias: do not paraphrase safety lifecycle terms loosely.

## SAE J3016

- Focus: definitions for driving automation levels and roles.
- Common terms: ODD, DDT, fallback-ready user, ADS, sustained lateral and longitudinal control.
- Translation bias: keep taxonomy and role definitions stable across the document.

## Practical Use

- Prefer the document's own glossary over this file when both exist.
- Prefer user-provided corporate terminology over this file when both exist.
- If an official Chinese rendering is uncertain, keep the English term plus acronym rather than inventing a confident but unstable translation.
